# excel-netflix-project
## Project Overview 
An exploratory data analysis of Netflix titles using Excel, featuring data cleaning, pivot tables, and an interactive dashboard.
## Dataset
- *Source:* Netflix Titles Dataset
- *Records:* 838 titles (810 Movies, 28 TV Shows)

## What I Did
- *Data Cleaning:* Split date_added into separate Month, Day, 
  and Year columns for better analysis
- *Pivot Tables:* Built multiple pivot tables to analyze:
  - Content by country
  - Movies vs TV Shows breakdown
  - Ratings distribution
  - Release year trends (2008–2021)
  - Top genres/categories
- *Dashboard:* Created a visual dashboard summarizing key insights

## Key Findings
- The United States dominates Netflix content (282 titles)
- India is the second largest contributor (112 titles)
- Movies outnumber TV Shows significantly (810 vs 28)
- TV-MA is the most common rating (242 titles)
- 2020 had the highest content additions (192 titles)

## Tools Used
- Microsoft Excel (Pivot Tables, Dashboard)

## File Structure
- netflix_titles — Raw dataset
- data clean — Cleaned and processed data
- PIVOT TABLE — Analysis pivot tables
- DASHBOARD — Visual summary dashboard
